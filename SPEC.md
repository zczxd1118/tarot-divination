# 塔罗占卜应用 - 规格说明书 (V2.0)

## 一、项目概述

### 1.1 项目名称
塔罗占卜应用（经典韦特体系）

### 1.2 产品目标
基于经典 Rider-Waite Tarot（韦特塔罗），构建一个具备：
- 双模式抽牌（选牌 / 随机）
- 多牌阵系统（单张/三张/五张）
- 强交互仪式感
- 真实塔罗牌画面展示
- 针对用户问题的深度解读

### 1.3 产品定位
- 娱乐 + 心理引导
- 非专业预测工具

---

## 二、核心功能模块

| 模块 | 说明 |
|------|------|
| 抽牌模式系统 | Manual（选牌）/ Random（随机）双模式 |

## 五、抽牌系统（双模式）

### 5.1 模式一：用户选牌（Manual）

**完整流程**:
1. 用户选择牌阵
2. 用户输入问题（可选）
3. 展示78张牌（背面朝上）
4. 鼠标 hover 高亮
5. 点击选择牌（根据牌阵数量）
6. **翻牌阶段** - 选满后显示选中牌（背面朝上），点击"翻牌"按钮
7. 依次翻牌揭示牌面
8. 进入结果页

**交互规则**:
- 根据牌阵数量选择对应张数
- 不允许重复选择
- 显示选择序号
- 选满后显示"命运已锁定"页面
- 翻牌按钮有呼吸动画效果

### 5.2 模式二：随机抽牌（Random）

**流程**:
1. 用户选择牌阵
2. 用户输入问题（可选）
3. 洗牌动画（2秒）
4. **抽牌阶段** - 依次抽出对应数量牌，所有牌保持背面朝上
5. **翻牌阶段** - 依次翻面，展示牌面+牌名+正逆位
6. 进入结果页

**设计目的**:
- 增强仪式感
- 模拟真实占卜流程
- 提升用户期待

## 六、正逆位规范

### 6.1 正位展示
- 正常显示塔罗牌图片
- 图片不进行任何翻转或旋转
- 文字信息正常显示

### 6.2 逆位展示
- **整张卡牌旋转180度**（不使用镜像翻转）
- 图片随卡牌一起旋转
- 文字信息保持可读方向
- 位置标签跟随卡牌旋转

### 6.3 选牌页面
- 选中卡片后，槽位只显示位置标签
- 牌名在翻牌揭示前保持隐藏
- 翻牌后显示完整牌面和信息

## 七、界面结构

### 7.1 首页
- 标题和副标题
- 抽牌模式选择（Manual / Random）
- **牌阵选择（单张/三张/五张）**
- 问题输入框（可选）
- 开始按钮

### 7.2 选牌页（Manual）
- 选牌槽位（根据牌阵数量动态生成）
- 78张牌网格
- 返回按钮

### 7.3 翻牌阶段
- 选中牌展示（背面朝上）
- 翻牌按钮
- 重新选牌按钮

### 7.4 结果页
- 牌阵模式和位置标签
- 牌面展示区
- AI解读区域
- 底部操作按钮
