# 塔罗占卜 · Tarot Divination

一个纯前端的韦特塔罗在线占卜应用，支持随机抽牌与手动选牌，包含 78 张韦特经典牌面与详细解读。

## ✨ 特性

- 🎴 78 张完整韦特塔罗牌（图片来自 jsDelivr CDN，本地 SVG 兜底）
- 🔮 支持单张 / 三张（过去-现在-未来）/ 凯尔特十字 等多种牌阵
- 🎲 随机抽牌 + 手动选牌两种模式
- 📜 历史记录持久化（localStorage）
- 📱 完美适配移动端、平板、桌面端
- 🌙 暗夜星空主题，纯静态零后端

## 🚀 在线访问

部署在 GitHub Pages：<https://你的用户名.github.io/tarot-divination>

## 💻 本地运行

```bash
# 任何静态服务器都可以
python3 -m http.server 8080
# 访问 http://localhost:8080
```

## 📦 部署

只有一个 `index.html`，可以部署到任何静态主机：

- GitHub Pages
- Cloudflare Pages
- Vercel / Netlify
- EdgeOne Pages
- 任何 Nginx / Apache
